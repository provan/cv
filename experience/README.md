# Опыт работы

Один файл на роль/компанию. Имя файла: `YYYY-YYYY_company_role.md` (отсортируется хронологически).

## Шаблон

```markdown
# {Роль} @ {Компания}

**Период:** YYYY-MM — YYYY-MM
**Роль:** {title}
**Команда / Орг:** {scale, размер команды, под кем/над кем}
**Стек:** {основной техстек одной строкой}

## Контекст

Что за компания, какой продукт, какая бизнес-задача. Масштаб (DAU, выручка, команда).

## Что делал

- Глагол + объект + влияние
- ...

## Результаты

- Измеримое: метрика → изменение
- ...

## Технологии

Полный список.
```
