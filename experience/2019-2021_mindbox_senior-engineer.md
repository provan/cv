## Контекст

Mindbox — платформа маркетинговой автоматизации (CDP/CRM). Совмещал роли разработчика, архитектора и техлида.

## Что делал

- Спроектировал и выпустил **Mobile SDK** (архитектура front + back + QC tooling, план миграции; контроль 3 команд разработки и миграции ключевых клиентов).
- Подготовил mailing-подсистему к Black Friday: проектировал и проводил load tests, искал bottlenecks, приоритизировал критичные оптимизации, делал scaling trade-offs.
- Ускорил внедрение **Cassandra** в компании: серия лекций, консультации команд по схеме данных, помощь с load tests и запуском.
- Спроектировал и реализовал **async-клиент для RabbitMQ** против thread starving в API gate под нагрузкой.
- Ускорил модуль сегментов на порядок: рефакторинг, нагрузочное тестирование новой SQL-схемы, миграция **1ТБ через 300+ БД**, бесшовное переключение.
- Спроектировал и внедрил автоматизированный контроль SLA.
- Разбирал и предотвращал критичные инциденты на MSSQL Server под высокой нагрузкой.

## Результаты

- **Mobile SDK:** 100M+ установок, стоимость интеграции для клиентов снижена **в 30 раз**.
- **Mailing подсистема:** выдержала Black Friday на нагрузке **~10k RPS**.
- **Cassandra** прижилась как стандарт для подходящих сценариев.
- Модуль сегментов: ускорение **на порядок** (10×+).
