# О себе

## Кто я

Возглавляю LLM Platform в Avito — один из крупнейших classifieds мира (50M+ MAU). Команда владеет всем AI-инфра-слоем: LLM Gateway, Eval, RAG, Agent SDK. Флагман — **Avi**, multi-agent buyer assistant: миллионы пользователей, 0.3% error rate, p75 latency 8s.

До Avito — 15 лет distributed systems, Kubernetes, high-load сервисов. Многоагентные системы по сути требуют того же, что и классический backend: state management, retries, observability. Только с LLM non-determinism сверху.

## Как я работаю

- **AI-native builder.** Агенты пишут код быстрее, чем я успеваю ревьюить — поэтому строю систему контроля, где проверяю только DoD, бизнес-кейсы и спецификации.
- **Specification Driven Development (SDD) + TDD.** Red → Green → Refactor выполняют агенты, я ставлю спеку и контролирую acceptance.
- **Acceptance tests как материализованные DoD.** Один тест — один assert, DSL поверх test-data-builder. Изменение и удаление тестов — только через мой явный апрув.
- **Конституция проекта.** У каждого моего проекта есть документ с критичными тезисами, который задаёт рамки решений.

## Что меня драйвит

- Превращение нечётких бизнес-задач в строгие архитектуры — особенно когда нужно выжать перформанс из high-load systems.
- Построение команд: за карьеру нанял десятки инженеров, в Ozon из одной команды вырастил три self-sufficient.
- Делать AI-инфраструктуру, на которой могут спокойно работать продуктовые команды, не разбираясь во внутренностях LLM-стека.
